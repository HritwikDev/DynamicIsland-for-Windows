# DynamicIsland-for-Windows

A modern **Dynamic Island-inspired desktop widget** built with **C# WinForms (.NET 9)** for Windows.

It provides a floating island at the top of your desktop with quick access to media controls, clipboard history, calendar, favorites, Bluetooth devices, file tray, and more.

---

---

# Features

* Floating Dynamic Island interface
* Smooth animations
* Media Controller
* Calendar widget
* Clipboard history
* Favorites launcher
* File Tray (Drag & Drop)
* Bluetooth device manager
* Activity Manager
* Plugin Support
* Auto Update Support
* Settings panel
* Windows 10 & Windows 11 support

---

# Built With

* C#
* .NET 9
* Windows Forms
* SQLite
* Velopack (Auto Updates)

# Getting Started

Clone the repository

```bash
git clone https://github.com/USERNAME/DynamicIsland.git
```

Move into the project

```bash
cd DynamicIsland
```

Restore packages

```bash
dotnet restore
```

Build

```bash
dotnet build
```

Run

```bash
dotnet run --project src/DynamicIsland/DynamicIsland.csproj
```
